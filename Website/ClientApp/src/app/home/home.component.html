Raid Extractor

Welcome to Raid: Shadow Legends - Artifact Optimizer, the website to optmize your artifacts in Raid: Shadow Legends.

All code used by this website and the artifact extractor application can be found here: https://github.com/LukeCroteau/RaidExtractor

You need to run the RaidExtractor application while also running Raid: Shadow Legends to extract all your hero and artifacts from the Windows version of the game.
